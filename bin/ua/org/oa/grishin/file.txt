Для выбора операции введите символ либо цифру операции исходя из требований на экране.
Перечень операций и обозначений в иерархическом порядке:
0 ---> Автомобиль (параметры по умолчанию)
1 ---> внести новый автомобиль;
	a ==> автомобиль (марка, год выпуска, стоимость)
	b ==> автомобиль (марка, год выпуска, стоимость, объем двигателя)
2 ---> автомобиль с пробегом;
	a ==> автомобиль (марка, год выпуска, пробег, стоимость)
	b ==> автомобиль (марка, год выпуска, пробег, объем двигателя, стоимость)
3 ---> выбрать вариант просмотра перечня автомобилей;
4 ---> выполнить фильтрацию по следующим параметрам:
	a ==> только новые автомобили
	b ==> только автомобили с пробегом
	c ==> по стоимости
	d ==> по году изготовления
	e ==> по цене 
5 ---> просмотреть автомобили чей пробег:
	a ==> выше среднего
	b ==> выше определенного значения в км;
	c ==> ниже среднего;
	d ==> ниже определенного значения в км;

m ---> отобразить перечень операций
q ---> выход.
